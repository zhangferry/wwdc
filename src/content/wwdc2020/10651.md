## 一句话判断

App Store Connect 在 2020 年迎来了一波实用更新——App Clips 的提交流程、新的 App 审核信息沟通方式、改进的销售和趋势报告，以及多用户角色管理，都是解决开发者日常痛点的功能。

## 这场 Session 讲了什么

这场 Session 概览了 App Store Connect 在 2020 年的各项改进，主要集中在以下几个方面：

首先是 App Clips 的管理。App Clips 是 iOS 14 的重点新功能，App Store Connect 新增了 App Clips 的提交和管理流程，包括如何配置 App Clip 的 URL 关联和 NFC 标签。

其次是审核沟通的改进。开发者现在可以直接回复审核团队的消息，审核状态的通知也改为了实时推送，不再是之前那种「提交后刷页面等状态变化」的体验。

第三是商业数据的增强。新的 Sales and Trends 报告加入了更多维度：订阅留存率、退款分析、按地区和设备类型的细分数据。API 层面，App Store Connect API 1.3 新增了订阅状态查询和客户管理能力。

最后是团队协作的改进，新增了更细粒度的角色权限管理。

## 值得深挖的点

**审核沟通终于不是单向的了**

以前被拒后，你只能修改后重新提交，无法和审核团队直接对话。2020 年的更新允许你在 Resolution Center 中直接回复审核团队，针对具体条款进行讨论。这对于被 Guideline 2.1（性能）或 3.1.1（内购）拒审的 app 来说，沟通效率大幅提升。你可以在消息中附上截图、视频、或者解释技术实现细节。

**App Clip 的提交流程**

App Clip 不是独立提交的，而是作为主 app 的附属品。在 App Store Connect 中，你需要为 App Clip 配置一个独立的 `NSAppClip` 字段的 Info.plist，设置关联的 URL 和显示标题。提交流程会自动检测 App Clip 的二进制大小（限制 10MB），超标会直接拒绝。

## 代码片段

```bash
# App Store Connect API 1.3 新增的订阅管理接口

# 查询订阅状态
curl -H "Authorization: Bearer $JWT" \
  "https://api.appstoreconnect.apple.com/v1/subscriptions/{subscriptionId}"

# 查看订阅定价点
curl -H "Authorization: Bearer $JWT" \
  "https://api.appstoreconnect.apple.com/v1/subscriptionPrices"

# 获取销售报告（改进后的格式）
curl -H "Authorization: Bearer $JWT" \
  "https://api.appstoreconnect.apple.com/v1/salesReports?filter[frequency]=DAILY&filter[reportType]=SALES&filter[reportSubType]=SUMMARY&filter[vendorNumber]=123456&filter[reportDate]=2020-06-22"
```

## 最佳实践

1. **提交 App Clip 时先在本地测大小**。用 Xcode 的 App Thinning Size Report 检查 App Clip 的安装大小。10MB 的限制包含了可执行代码、资源和 Assets Catalog，但不算系统框架。建议控制在 8MB 以内留点余量。
2. **利用 Resolution Center 积极沟通**。被拒后不要急着修改代码重提。先仔细阅读审核意见，如果有疑问就在 Resolution Center 里问清楚。很多情况下是误解，一轮沟通就能解决。
3. **定期拉取销售报告做分析**。App Store Connect API 支持自动拉取 Sales and Trends 报告。建议写个定时任务每天拉取，导入自己的分析系统，比在网页上翻图表高效得多。

## 还有什么值得关注

- App Store Connect 现在支持为不同地区设置不同的发布日期，做灰度发布更方便了。
- 新增了「App 审核申诉」快捷通道，如果你认为审核决定有误，可以直接在 Resolution Center 中发起正式申诉。
- 订阅类 app 可以在 App Store Connect 中配置 introductory offer（首月优惠）和 promotional offer（老用户回归优惠），价格策略更灵活。
