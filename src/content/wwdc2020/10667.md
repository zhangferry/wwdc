## 一句话判断

Apple Business Manager 现在支持「自定义 App」分发——你可以把 app 定向分发给特定企业客户，不上公开 App Store，不需要企业证书，通过 Apple Business Manager 的私有商店完成交付。

## 这场 Session 讲了什么

这场 Session 讲解了如何通过 Apple Business Manager（ABM）实现企业级 app 的定制分发。

传统的企业分发有两种方式：公开 App Store（所有人可见）和企业证书（Enterprise Certificate，需要每年续费且仅限企业内部使用）。2020 年苹果提供了第三种方式：Custom App Distribution。

Custom App 允许你在 App Store Connect 中上传 app，但只在 Apple Business Manager 中对指定的组织可见。指定的组织通过 ABM 批量购买并分发给自己员工。这个流程有完整的 App Review 审核（保证质量），但不会出现在公开 App Store 的搜索结果中。

Session 还介绍了 ABM 的 Content Manager 功能，用于管理分发权限、批量购买和设备分配。

## 值得深挖的点

**Custom App vs Enterprise Certificate 的选择**

Enterprise Certificate（299 美元/年）的问题是：app 不经过 App Review、证书到期后 app 会崩溃、只限企业内部使用。Custom App 走 App Review 审核，有完整的 App Store 基础设施（版本管理、更新推送、反馈收集），但只能通过 ABM 分发给指定组织。如果你的客户是外部企业（B2B SaaS），Custom App 是更好的选择。

**分发范围的控制**

在 App Store Connect 中，你可以为 Custom App 设置「可分发组织」的白名单，通过 Organization ID（DUNS Number）精确控制哪些企业能看到和购买你的 app。未在白名单中的组织完全看不到这个 app 的存在。

## 代码片段

```bash
# App Store Connect API 管理 Custom App 分发

# 查询 app 的分发类型
curl -H "Authorization: Bearer $JWT" \
  "https://api.appstoreconnect.apple.com/v1/apps/{appId}/appInfoLocalizations"

# 设置 app 为 Custom App 分发模式
# 在 App Store Connect 网页端操作：
# 1. 进入 app 的 Pricing and Availability
# 2. 选择 "Custom App Distribution"
# 3. 添加目标组织的 DUNS Number
# 4. 提交审核

# 批量购买 API
curl -H "Authorization: Bearer $JWT" \
  "https://api.appstoreconnect.apple.com/v1/apps/{appId}/customAppOrganizationId"
```

## 最佳实践

1. **提前获取客户的 DUNS Number**。在设置 Custom App 分发时需要目标企业的 Organization ID，这基于 DUNS Number。和客户对接时第一时间确认这个信息，避免发布流程被卡住。
2. **给 Custom App 做好版本管理**。企业客户不像普通消费者那样频繁更新 app。每次发版要做好向下兼容，至少支持前两个主要版本。在 release notes 中明确标注变更内容。
3. **利用 ABM 的批量购买接口做自动化**。如果你有很多企业客户，通过 App Store Connect API 批量管理分发权限比在网页上一个一个操作高效得多。

## 还有什么值得关注

- Apple School Manager（ASM）也支持 Custom App 分发，教育场景和商业场景使用相同的流程。
- Custom App 同样支持内购和订阅，收入分成比例与公开 App Store 相同。
- 2020 年苹果还简化了 ABM 的注册流程，小企业不再需要 100 台设备才能注册。
