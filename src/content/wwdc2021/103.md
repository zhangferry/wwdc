## 一句话判断

每年的 ADA 获奖名单就是一份「什么才是好的 iOS App」的标准答案——仔细研究这些获奖作品比看任何设计教程都有用。

## 这场 Session 讲了什么

Apple Design Awards（ADA）是 WWDC 期间最让人期待的环节之一，表彰在设计、技术实现和用户体验上表现卓越的 App 和游戏。2021 年的颁奖典礼在 WWDC 第二天举行，获奖作品横跨多个类别。

2021 年的 ADA 评选标准分为六个维度：视觉与图形（Visuals and Graphics）、交互与反馈（Interaction and Feedback）、包容性（Inclusivity）、社会影响（Social Impact）、愉悦与趣味（Delight and Fun）、创新思维（Innovation）。每个维度对应一个或多个获奖作品。苹果不再按「最佳 App」「最佳游戏」这样的传统类别颁奖，而是按设计维度评选——这意味着一个 App 可能因为出色的包容性设计获奖，而它的视觉设计可能并不是最出色的。

颁奖典礼由苹果的设计团队主持，播放了每个获奖作品的幕后制作纪录片，展示开发者如何从一个想法出发，经过反复打磨最终呈现出色的体验。这些纪录片本身就是很好的设计教育材料。

## 值得深挖的点

**2021 年部分获奖作品的分析。** 获奖名单中涵盖了不同类型的 App 和游戏：从创意工具到生产力 App，从独立游戏到大型 3A 作品。苹果选择这些作品的原因各不相同——有的因为突破性的交互设计（比如利用 Apple Pencil 的压感和倾斜来做完全不同的创作体验），有的因为出色的辅助功能支持（比如为视障用户提供完整的音频导航），有的因为在社会影响方面的贡献（比如帮助用户减少碳排放或改善心理健康）。研究这些获奖作品的一个方法是：下载每一个，用 VoiceOver 打开它，关掉声音只用视觉操作它，把字体调到最大——你会发现获奖作品在这些极端场景下依然可用。

**六个评选维度的设计启示。** 苹果用这六个维度定义了「好的设计」的标准，开发者可以用它们作为自查清单。视觉与图形不只是「好看」，而是视觉风格是否服务于功能和品牌；交互与反馈不只是「能用」，而是操作是否有即时、清晰的反馈；包容性不只是「支持 VoiceOver」，而是是否让不同能力的用户都感到被重视；社会影响不只关注功能本身，还关注 App 是否帮助用户成为更好的人；愉悦与趣味是最主观但也是最能区分优秀和卓越的维度——那些让你忍不住微笑的细节；创新思维则奖励那些敢于挑战常规的设计决策。

    var body: some View {

struct AccessibleCustomView: View {

        CustomProgressView(value: value)

            .accessibilityHint("正在下载文件")
    }
}

## 最佳实践

- 把 ADA 的六个评选维度当作你的设计审查清单——每次版本发布前对照检查
- 研究获奖作品时不要只看表面，要分析它们在极端场景下的表现（VoiceOver 开启、最大字号、高对比度模式）
- 设计创新不一定需要复杂的技术——很多获奖作品的亮点是「用简单的方式解决了一个被忽视的问题」
- 关注每年的 ADA 获奖名单，理解苹果的设计价值观在如何演变

## 还有什么值得关注

- ADA 2021 的完整获奖名单可以在 Apple Developer 网站上查看
- 往年的获奖作品（从 1997 年开始）都保存在 Apple Design Awards 页面中，是很好的设计参考库
- 苹果在 HIG（Human Interface Guidelines）中引用了很多 ADA 获奖作品作为设计范例
