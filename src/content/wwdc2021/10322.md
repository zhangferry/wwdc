## 一句话判断

周二是 Session 密集轰炸的第一天——Xcode Cloud 三连讲、SF Symbols 3、MusicKit 是今天的技术干货。

## 这场 Session 讲了什么

WWDC21 第二天的每日回顾，约 2 分钟。周一是发布日，周二是深入讲解的第一天。这一天发布了大量的技术 Session，涵盖开发者工具、框架 API 和设计指南。

重点 Session 包括：Xcode Cloud 的三连讲（Meet / Explore / Customize），完整介绍了苹果的 CI/CD 解决方案；SF Symbols 3 带来了新的 Rendering Modes 和 3000+ 图标；MusicKit for Swift 把 Apple Music API 包装成了类型安全的 Swift 原生框架；包容性设计的两个 Session（practice 和 process）从不同角度讲解了无障碍设计。

对于 Swift 开发者来说，这一天还有关于 Swift Concurrency 的深入讲解——async/await 的设计原理、Actor 的使用模式、以及如何用 Sendable 保证并发安全。

## 值得深挖的点

周二是 WWDC 中信息密度最高的一天。如果你的时间有限，优先看 Xcode Cloud 的第一讲（了解苹果 CI 的全貌）和 MusicKit for Swift（如果你做音乐相关的 App）。SF Symbols 3 的变化直接影响 UI 设计，值得所有开发者关注。

## 代码片段

```swift
// 周二发布的 MusicKit for Swift——类型安全的 Apple Music API
import MusicKit

func searchAndPlay() async throws {
    // 搜索歌曲
    var request = MusicCatalogSearchRequest(term: "Bohemian Rhapsody", types: [Song.self])
    let response = try await request.response()

    guard let song = response.songs.first else { return }

    // 直接用系统播放器播放
    let player = MPMusicPlayerController.applicationQueuePlayer
    player.setQueue(with: [song.id.rawValue])
    player.play()
}
```
