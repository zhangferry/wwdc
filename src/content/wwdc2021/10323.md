## 一句话判断

周三的重心是健康、辅助功能和系统服务——如果你的 App 涉及 HealthKit、Core Motion 或无障碍适配，今天的 Session 不要错过。

## 这场 Session 讲了什么

WWDC21 第三天的每日回顾，约 2 分钟。周三是「底层能力日」——大量 Session 聚焦在框架层的 API 更新。

这一天的重要内容包括：「Measure health with motion」讲解了 Core Motion 和 HealthKit 在运动健康场景中的深度应用；watchOS 的辅助功能更新（AssistiveTouch）让上肢运动障碍用户也能操作 Apple Watch；「Build a research and care app」系列继续，第三部分讲了健康数据的可视化设计；Shortcuts 和 Siri 的 Action 设计指南教你如何让系统理解你的 App 能做什么。

此外，关于 UIKit 新增的配置 API、StoreKit 2 的早期预告、以及 privacy 相关的 API 更新也在这一天有专门的 Session。

## 值得深挖的点

周三的内容偏底层和专项领域。如果你的 App 涉及健康数据，HealthKit 的 `HKWorkoutBuilder` 实时数据流 API 值得仔细研究。如果你做智能家居，Matter 协议的接入指南也在这一天发布。

- 健康类 App 的开发者优先看 HealthKit 相关的三个 Session（数据采集、存储、可视化）
- 如果你还没做辅助功能适配，今天的辅助功能相关 Session 是最好的入门
