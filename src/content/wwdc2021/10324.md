## 一句话判断

周四是图形、音频和媒体的重头戏——Metal bindless rendering、Spatial Audio、AVKit 更新全部在这一天释放。

## 这场 Session 讲了什么

WWDC21 第四天的每日回顾，约 2 分钟。周四的重心是图形渲染、音频处理和媒体框架的更新。

关键 Session 包括：Metal 的 bindless rendering 深入讲解，展示如何用 Argument Buffers 和 Heap Resources 消除 draw call 的 binding 瓶颈；Spatial Audio 的接入指南教开发者如何用 AVAudioEnvironmentNode 实现 3D 空间音效；AVKit 的新功能让 `AVPlayerViewController` 终于支持自定义控制按钮；Apple Music API 和 MusicKit for Swift 的实战教程也在这一天发布。

非技术方面，「Out of this world... on to Mars」是 NASA JPL 工程师的励志演讲，Apple Design Awards 颁奖典礼也在周四举行。

## 值得深挖的点

如果你是游戏开发者，周四的 Metal bindless rendering Session 是必看内容。如果你的 App 有音频播放，Spatial Audio 的接入成本极低但体验提升明显。AVKit 的自定义按钮功能让你终于不用重写整个播放器 UI。

## 代码片段

```swift
// 周四发布的 AVKit 自定义按钮功能
import AVKit

let playerVC = AVPlayerViewController()
playerVC.player = AVPlayer(url: videoURL)

// 终于可以在系统播放器上加自定义按钮了
let pipAction = AVPlayerViewController.CustomAction(
    image: UIImage(systemName: "pip.enter")!,
    accessibilityLabel: "画中画"
) {
    enterPictureInPicture()
}

playerVC.customActions = [pipAction]
```
