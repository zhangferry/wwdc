这是 Apple Design Awards（Session 103）的美国手语翻译版本——内容和原版一致，画面中增加了 ASL 翻译员的实时手语翻译。

## 这场 Session 讲了什么

这是 WWDC21 Apple Design Awards 颁奖典礼的无障碍版本，为听障开发者提供 ASL 手语翻译。内容与 Session 103 完全一致。

2021 年的 Apple Design Awards 按六个维度评选：视觉与图形、交互与反馈、包容性、社会影响、愉悦与趣味、创新思维。颁奖典礼由苹果设计团队主持，播放了每个获奖作品的幕后制作纪录片。

获奖作品展示了不同的设计哲学和技术实现方式：有的以突破性的交互设计获奖，有的因为出色的辅助功能支持而受到表彰，还有的因为在社会影响方面的贡献而脱颖而出。这些作品共同体现了苹果对「好的 App 设计」的定义——不只是视觉上好看，而是从每个用户的角度都能提供出色的体验。

## 值得深挖的点

六个评选维度提供了一个实用的设计自查框架。无论你的 App 规模大小，都可以用这六个维度做一轮自我审视。特别是「包容性」和「社会影响」这两个维度——在之前的 ADA 评选中权重较低，但从 2021 年开始被提升到了和视觉效果同等重要的位置。这反映了苹果设计价值观的演变：好的设计必须对所有人好。

## 代码片段

```swift
// ADA 获奖作品通常具备的技术特质
// 完整的辅助功能支持是基本要求

struct AccessibleButton: View {
    let title: String
    let action: () -> Void

    var body: some View {
        Button(action: action) {
            Text(title)
                .font(.body)  // 支持动态字体
                .padding(12)
                .frame(maxWidth: .infinity)
                .background(Color.blue)
                .foregroundColor(.white)
                .cornerRadius(10)
        }
        .frame(minHeight: 44)  // 满足最小点击区域
        .accessibilityElement(children: .combine)
    }
}
```

- 把 ADA 的六个维度纳入你的设计审查流程
