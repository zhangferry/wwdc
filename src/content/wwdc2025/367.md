## 一句话判断

10 分钟看完 WWDC25 全部重点 -- Liquid Glass、Foundation Models、Xcode 26、Swift 6.2、SwiftUI 富文本和性能、Metal 4、visionOS 沉浸式媒体。适合团队分享或快速复习，不适合用来学技术细节。

## 这场 Session 讲了什么

这是一场高度浓缩的 recap，将 PSOTU 近一小时的内容压缩到约 10 分钟，用快速剪辑的方式回顾了 WWDC25 所有核心技术更新：

**Liquid Glass**：全新的动态材质设计，结合玻璃光学特性和流体感。Icon Composer 支持模糊、半透明、高光测试和多模式预览。

**Foundation Models 框架**：端侧优化的基础模型，支持文本提取、摘要等功能。数据不出设备，无费用，离线可用。

**Xcode 26**：内置 ChatGPT 和第三方模型接入。Coding Tools（代码版 Writing Tools）提供 preview/playground/fix 等建议操作。支持对话历史。

**Swift 6.2**：性能、并发、互操作三大主题。MainActor by default 简化单线程代码。

**SwiftUI**：新 web APIs、3D Charts、富文本编辑（`AttributedString` binding）、性能大幅提升。

**Metal 4**：MetalFX 帧插值和降噪。Game Porting Toolkit 3 支持 Windows upscaling 技术。PlayStation VR2 Sense 控制器支持。

**快速轮播**：Terminal 24-bit 色彩和新主题、visionOS 180/360 度沉浸式媒体（APMP）、CarPlay Live Activities、declared age range API、Accessibility Nutrition Labels、Family Sharing 更新、Assistive Access 改进。

## 值得深挖的点

1. **这场 recap 的价值在于"一张图"**。10 分钟建立 WWDC25 的完整认知地图，然后你根据自己的技术栈去挑选具体 Session 深入。不要用它替代具体 Session。

2. **recap 中提到的每个技术点都有对应的专门 Session**。这是选择后续学习路线的最佳索引。

3. **对于团队 lead 或 tech lead**，这是做技术分享的好素材。放给团队看 10 分钟，然后根据项目需要分配具体 Session 给不同成员。

## 最佳实践

1. **用 recap 建立全局认知，用具体 Session 深入技术细节**。推荐观看顺序：Keynote -> PSOTU -> Recap -> 挑选具体 Session。

2. **根据你的技术栈选择深入方向**：
   - UI/设计：Liquid Glass、Design System、SF Symbols 7
   - AI/ML：Foundation Models
   - 工具链：Xcode 26、Swift 6.2、UI Automation
   - 游戏/图形：Metal 4
   - 平台：watchOS 26、visionOS 26

3. **recap 中的"快速轮播"部分是容易被忽略的宝藏**。Terminal 更新、CarPlay Live Activities、declared age range API 等都是小而美的改进，可能正好解决你的某个痛点。
